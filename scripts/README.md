# 折立印名片套版助手

脚本猫 / Tampermonkey 用户脚本，用在 `diy.zheliyin.com` 设计器页面。

## 0.3.0 重点

- 支持追加信息：新粘贴一段客户资料后点“追加信息”，会叠加到原字段，不会清空。
- 精简按钮：去掉“默认布局”“留白布局”“扫描图层”“背面业务”。
- “填背面”放在“识别并填正面”旁边。
- 电话、微信、邮箱、网址、地址都支持多条，例如电话1、电话2、地址1、地址2。
- 正面会保证文字里实际存在的字段都被放进画面；原文字没有的字段不会填。
- 当前模板文字图层不够时，会自动新增文字图层。
- 自动清理脚本自己生成的无关旧图层和常见空占位图层。
- 自动优化排版，最小字号为 14。
- 脚本内置 GitHub 更新检测，有新版时会弹出提示。

## 使用

1. 在脚本猫或 Tampermonkey 中新建脚本。
2. 粘贴 `zheliyin-card-assistant.user.js` 的全部内容并保存。
3. 打开折立印设计器页面。
4. 粘贴客户资料，点“识别并填正面”。
5. 如果又收到补充资料，粘贴新资料后点“追加信息”。
6. 需要背面业务范围时切到背面，再点“填背面”。

## 匹配页面

```text
https://diy.zheliyin.com/diyWeb/third/*
https://diy.zheliyin.com/diyWeb/third/19792684/1/thirdLoginDiyEdit.do
https://diy.zheliyin.com/diyWeb/third/1195351/2114747/999/thirdDiyAdd.do
```

## 字段结构

- 中文公司
- 英文公司
- 姓名
- 职位
- 电话，可多条
- 微信，可多条
- 邮箱，可多条
- 网址，可多条
- 地址，可多条
- 主营业务，可多条

API Key 保存在脚本猫 / Tampermonkey 的本地存储里，不写在脚本文件中。
